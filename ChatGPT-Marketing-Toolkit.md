- Practical tips audience can use

# Tips for Better Results

1. **Be specific** — The more context you give, the better the output
2. **Iterate** — First draft is never final. Refine with follow-up prompts
3. **Add constraints** — Word count, tone, format preferences
4. **Use examples** — Show ChatGPT what "good" looks like
5. **Combine prompts** — Stack multiple prompts for complex content

---

# Quick Reference: Prompt Anatomy

```
[Role/Context] + [Task] + [Requirements] + [Format] = Great Output

Example:
[Act as a direct response copywriter] + [Write 3 follow-up emails] + 
[that remind prospects about our service without being pushy] + 
[Keep each email under 100 words with a clear CTA]
```
