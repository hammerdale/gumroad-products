# OpenClaw Real Estate Agent Workflow Package

## What's Included

This is a complete, pre-configured OpenClaw workflow designed specifically for real estate agents. It automates lead follow-up, property updates, and client communication.

## Files in This Package

1. `system-prompt.md` - The core instructions for your AI agent
2. `skills.yaml` - Tool permissions and capabilities
3. `workflow-config.yaml` - Automation settings
4. `setup-guide.md` - Step-by-step installation instructions

## What It Does

✅ **Lead Management**
- Responds to new leads within minutes
- Qualifies prospects with scripted questions
- Schedules follow-up calls automatically

✅ **Property Updates**
- Sends new listings to matching buyers
- Notifies agents of price changes
- Shares open house details

✅ **Client Communication**
- Follow-up sequences after showings
- Market update newsletters
- Birthday/anniversary greetings

## Quick Start

1. Install OpenClaw on your machine
2. Copy the files to your OpenClaw config directory
3. Configure your API keys (OpenAI/Anthropic, Discord, Email)
4. Run the workflow

See `setup-guide.md` for detailed instructions.

## Requirements

- OpenClaw installed and running
- OpenAI API key or Anthropic API key
- Discord bot (for agent communication)
- Gmail account or SMTP for email (optional)
