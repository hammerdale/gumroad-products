# Sales Pipeline Manager - Notion Template

A comprehensive CRM template for freelancers and sales professionals to track leads, deals, and client relationships.

## What's Included

### 1. Dashboard
- Total pipeline value
- Deals by stage chart
- Upcoming follow-ups
- Win rate percentage
- Monthly revenue tracker

### 2. Leads Database
- Name, company, email, phone
- Lead source (referral, cold outreach, ads, etc.)
- Status (New, Contacted, Qualified, Proposal Sent, Won, Lost)
- Estimated value
- Tags (industry, product interest)
- Notes (first contact info, needs)

### 3. Deals Pipeline
- Deal name
- Associated lead/client
- Stage (Discovery, Demo, Proposal, Negotiation, Closed Won, Closed Lost)
- Value/Amount
- Expected close date
- Probability (%)
- Next action
- Owner

### 4. Follow-up Reminders
- Linked to leads/deals
- Due date
- Reminder type (call, email, meeting)
- Status (pending, done, snoozed)

### 5. Client Directory
- Past clients from closed-won deals
- Contact details
- Past purchases
- Relationship notes

### 6. Scripts & Resources
- Email templates (follow-up, proposal, rejection)
- Call scripts
- Discovery questions
- Objection handlers

## How to Use
1. Duplicate the template to your Notion
2. Customize property options to fit your process
3. Add your leads and deals
4. Use the dashboard to track progress
5. Set up weekly reviews
